Stock Price Checker

This is a web app that return up-to-date stock prices.

Get single price and total likes

Compare and get relative likes

Usages

GET request example usage

[your_url]/api/stock-prices?stock=GOOG [your_url]/api/stock-prices?stock=GOOG&like=true [your_url]/api/stock-prices?stock=GOOG&stock=MSFT [your_url]/api/stock-prices?stock=GOOG&stock=MSFT&like=true

Example return

GET single price and total likes return:

{ "stockData": { "stock": "GOOG", "price": 786.90, "likes": 1 } }

GET compare and get relative likes return:

{ "stockData": [ { "stock": "MSFT", "price": 62.30, "rel_likes": -1 }, { "stock": "GOOG", "price": 786.90, "rel_likes": 1 } ] }
